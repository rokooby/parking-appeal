# Parking Appeal Verification

GenLayer smart contract for verifying parking ticket appeals using AI analysis.

- **Contract**: `backend/parking-appeal.py`
- **App**: https://rokooby.github.io/parking-appeal/

## Features

- AI-powered appeal analysis
- Evidence verification on blockchain
- Immutable appeal records
- Automated decision support

## Tech Stack

- **Backend**: GenLayer (Python)
- **Frontend**: React + TypeScript + Vite
- **Blockchain**: GenLayer Network
